
# _rca_ is a rich/RPN (and more) calculator

_rca_ is simple yet versatile.

 - it's natively a reverse polish notation calculator

 - it can evaluate infix-style expressions

 - it can run in either floating point or integer modes

 - it has the usual trigonometric and mathematical functions
     common to a scientific calculator

 - the integer word width can be set anywhere from two bits up
     the native width (probably 64)

 - it has all the usual bitwise operators

 - it's written in C, and maintained in a single source file


See the
[man page](html/rca-man.html)
and rca's
